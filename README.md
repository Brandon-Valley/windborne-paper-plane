# Windborne Paper Plane

A phone-first paper airplane arcade game controlled by wind.

## How to play

- Drag anywhere on the screen to set the wind direction.
- Blow into the phone microphone to set wind strength.
- Use the Silent Gust button or the spacebar as a fallback.
- Fly through rings, dodge storm clouds and birds, and chain clean gates for bonus points.

## Files

- `index.html` is the complete game. There is no build step.
- `.github/workflows/pages.yml` deploys the site to GitHub Pages.
